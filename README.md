# これはなに

最高のアニメ「[http://www.kinmosa.com/](きんいろモザイク)」のアリスのGIFアニメを表示するChrome拡張です。

# 使い方

話数とNoを指定するとその話数のGIFを表示します。  
IDを指定すると画像をIDで指定します。  
それぞれのパラメータの詳細については[http://mogashi.hateblo.jp/entry/2013/09/24/000619](API作者さんのブログ)を参照してください。
